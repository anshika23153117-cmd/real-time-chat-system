## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS, Axios, Socket.io-client
- **Backend:** Node.js, Express, Socket.io
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Frontend), Render (Backend)

## ✨ Features
- Real-time messaging using WebSockets.
- User Authentication (Registration & Login).
- Online/Offline status indicators.
- Responsive design for mobile and desktop.

## ⚙️ Local Setup
1. Clone the repo: `git clone https://github.com/anshika23153117-cmd/real-time-chat-system.git`
2. Install dependencies:
   - `cd chat-frontend && npm install`
   - `cd ../chat-backend && npm install`
3. Create a `.env` in the backend folder and add your `MONGODB_URI` and `JWT_SECRET`.
4. Run both servers: `npm start` (frontend) and `node server.js` (backend).
